AI DATA PROTOCOL: Viewport Debugger Dataset (v3.4)
1. DATASET PURPOSE
This dataset contains high-fidelity, lossless visual state captures from a Windows-based environment. It is designed for temporal state analysis, event detection, and real-time debugging where video compression artifacts must be avoided.
2. DIRECTORY STRUCTURE & INDEXING
Data is organized into discrete Session Folders to prevent temporal leakage.

    Root Directory: Session_[YYYYMMDD]_[HHMMSS]\
    Primary Index: metadata.csv (UTF-8)

2.1 Metadata Schema
Column	Type	Description
FrameIndex	Integer	Sequential ID (1-indexed). Use for primary sorting.
FileName	String	Relative path to the image asset.
TimestampMS	Integer	Precise milliseconds elapsed since session start (T+0).
3. IMAGE SPECIFICATIONS (TENSOR COMPATIBILITY)
The visual assets are pre-processed for direct ingestion into Vision Transformers (ViT) or Convolutional Neural Networks (CNN).

    Encoding: Lossless 24-bit RGB PNG.
    Color Space: Standard RGB (Channel 0: Red, Channel 1: Green, Channel 2: Blue).
    Channel Alignment: Internal hardware-level BGRA-to-RGB conversion has been applied. No additional channel swapping is required.
    Spatial Consistency: The Region of Interest (ROI) is static for the duration of the session. Coordinates are fixed based on the initial user-defined viewport.

4. INFERENCE GUIDELINES FOR AI SYSTEMS

    Temporal Analysis: To determine velocity or state change, compare Frame[N] with Frame[N-1]. Refer to TimestampMS in metadata.csv to calculate the exact delta_t between observations.
    No In-Frame Metadata: Images do not contain overlays, watermarks, or burnt-in timestamps. All telemetry is located in the external CSV.
    Resolution: Native capture resolution is maintained. Downscaling should be performed as a pre-processing step if required by specific model input layers.

5. HARDWARE CONTEXT
Captured via DirectX 11 Desktop Duplication API. The pipeline utilizes a triple-buffered RAM-to-NVMe architecture, ensuring that the TimestampMS reflects the actual GPU present-time with negligible jitter (<1ms).
